# 🎧 Digital Filter Design and Analysis (MATLAB)

## 📌 Overview
This project focuses on the design and analysis of digital systems and filters using MATLAB.  
It includes audio signal processing, echo system modeling, digital filter design, and frequency transformations.

---

## 🎯 Key Tasks

### 🔹 Audio Signal Analysis
- Performed time-domain analysis of an audio signal  
- Computed frequency spectrum using FFT  
- Verified signal energy using Parseval’s theorem  

### 🔹 Digital Echo System
- Modeled a digital echo system using delayed and scaled impulses  
- Designed an inverse system (equalizer) to remove echo distortion  
- Evaluated system performance using Mean Square Error (MSE)  

### 🔹 Digital Filter Design
- Designed and analyzed IIR Low-Pass Filters:
  - Butterworth  
  - Chebyshev Type I  
  - Chebyshev Type II  
  - Elliptic  
- Compared filters based on:
  - Mean Square Error (MSE)  
  - Power loss percentage  

### 🔹 Frequency Transformation
- Converted Low-Pass Filter (LPF) into:
  - High-Pass Filter (HPF)  
  - Band-Pass Filter (BPF)  
- Applied pole-zero rotation techniques  

---

## 🛠️ Tools
- MATLAB

---

## 📊 Results
- Successfully analyzed real audio signals in time and frequency domains  
- Implemented and evaluated digital echo and equalizer systems  
- Designed multiple IIR filters and compared their performance  
- Identified Chebyshev Type II filter as the best based on MSE and power efficiency
